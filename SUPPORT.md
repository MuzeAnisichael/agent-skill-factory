# Support

Agent Skill Factory is an early-stage open-source project.

## Questions

Use GitHub Discussions if enabled. Otherwise, open an issue with the `question` label.

## Bugs

Open a bug report and include:

- Command run.
- Expected behavior.
- Actual behavior.
- Operating system.
- Python version.
- Minimal Skill package or fixture that reproduces the issue.

Do not include private documents, credentials, customer data, or production Agent traces.

## Security

For security-sensitive reports, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.
