# Code of Conduct

## Our Pledge

We aim to make Agent Skill Factory a respectful, practical, and technically rigorous open-source project.

## Expected Behavior

- Be direct and constructive.
- Focus criticism on code, docs, tests, and design tradeoffs.
- Assume contributors may have different experience levels and time zones.
- Report security issues through private channels.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information without consent.
- Introducing malicious code, unsafe generated Skills, or real secrets.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or contributions that violate this code of conduct. Severe or repeated violations may lead to blocking from the project.
